MELHORIAS FEITAS NO PROJETO

1. Logo:
- Removido o excesso de espaço transparente ao redor da imagem.

- Tamanhos ajustados para tablet e celular.

Para instalar as dependências do Node.js:
npm install

Para executar o projeto:
npm start
ou o comando definido no package.json.
